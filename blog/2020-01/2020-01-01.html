通过对一个值使用 typeof 操作符会返回一下结果：

  • ‘undefined’：这个值未定义；
  • ‘boolean’： 这个值是布尔值；
  • ‘string’： 这个值是字符串；
  • ‘number’：这个值是数值；
  • ‘object’：这个值是对象或 null;
  • ‘function’：这个值是函数；

但是在实际应用中往往需要判断一个值是不是数组，这时候就用到了 instanceof，我们可以使用以下方法判断

var arr1 = [1,2,3];
console.log(arr1 instanceof Array); // true 说明arr1是数组

当然以上方法有存在漏洞，当你的项目中有多个fram，你穿梭其中的时候，由于它们的执行环境不同，跨frame实例化的对象彼此是不共享原型链的，因此导致上述检测代码失效

下面有一种最为简单有效的方法，废话不多说，直接上代码了：

var arr = [1,2,3];
console.log(toString.call(arr)==='[object Array]'); // true 则说明是数组
当然这个方法也适用于判断其他的数据类型：

var str = '123';
console.log(toString.call(str)); // '[object String]'
 
var num = 908;
console.log(toString.call(num)); // '[object Number]'
 
var func = function(){};
console.log(toString.call(func)); // '[object Function]'
 
var strNull = null;
console.log(toString.call(strNull)); // '[object Null]'
 
var strBoolean = true;
console.log(toString.call(strBoolean)); // '[object Boolean]'
